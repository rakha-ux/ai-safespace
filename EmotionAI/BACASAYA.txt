# Chatbot + Emotion AI Module Documentation

## Tujuan Modul

Modul ini bertugas untuk:

1. Menganalisis emosi pengguna dari jurnal dan pesan chat menggunakan Emotion AI.
2. Membangun konteks percakapan berdasarkan:

   * Jurnal terakhir pengguna.
   * Emosi pesan saat ini.
   * Riwayat percakapan.
3. Menghasilkan respon AI menggunakan Gemini.
4. Menyimpan log hasil deteksi emosi untuk evaluasi model.

# Menghilangkan Debug Translation

Saat ini translator dapat menampilkan:

```text
=== TRANSLATION ===
I am afraid of my thesis defense.
```

Output ini hanya untuk debugging.

Untuk menghapusnya:

Buka:

```python
emotion_ai/mentalhealthanalyzer.py
```

Cari:

```python
print("=== TRANSLATION ===")
print(translated_text)
```

Lalu hapus atau comment.
