Pada root project lakukan ini

python -m venv .venv

Aktifkan

.venv\Scripts\activate

Kalau berhasil biasanya muncul:

(.venv) C:\AI_SafeSpace>

lalu lakukan ini untuk melakukan instalasi

pip install -r requirements.txt

jika tidak bisa, install semua yang ada di requirements.txt seperti ini
pip install google-genai
pip install streamlit
pip install pandas
pip install numpy
pip install scikit-learn
pip install joblib
pip install python-dotenv
pip install deep-translator
pip install nltk
